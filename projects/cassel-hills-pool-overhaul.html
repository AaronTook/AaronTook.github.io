Cassel Hills Pool Unofficial Website

Project Description

This project was originally developed as a hobby project to take existing information about Cassel Hills Pool in Vandalia Ohio (mostly found on the current webpage) and reorganize the information, and create a custom web design for the site. Upon completing this project, I reached out to a contact on the City of Vandalia's website, since I suspected that they would not want an unofficial page showing up in the search results (which at the time it was doing quite well). As expected, they requested that I remove the site from search results, and that has been done, but a private version of the website, accessible only by knowing the URL, still exists.Information used to build the website was found from official City of Vandalia sources, but may be out of date depending on when this project was last updated.
