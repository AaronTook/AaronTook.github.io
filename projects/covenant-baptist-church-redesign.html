Covenant Baptist Church Unofficial Website

Project Description

After visiting Covenant Baptist Church's official website, I immediately wanted to see what I could do to update the feel of the site. This project emerged from that concept, with much of the site's content being either re-organized, updated, or both. This completed project can be visited via this link (https://aarontook.github.io/covenant-baptist-church).

Project Requirements

Technologies used in this project include the following:

  • HTML 5
  • CSS 3
  • JavaScript
  • W3.CSS (By W3schools.com)
