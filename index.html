Flask-Validate
Python, Flask, Flask-Mail, hashlib, secrets, HTML, W3.CSS

A Python and Flask system for validating possession of a email address or cell phone number. Designed to accept any email address and most major United States cell phone providers, this system will send an email with a validation code to the user, which when entered within five minutes of the send time, will pass. An incorrect or expired validation code will fail.
